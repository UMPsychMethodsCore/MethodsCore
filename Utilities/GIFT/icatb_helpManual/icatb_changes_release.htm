Changes from the release GroupICATv2.0b

  • Enabled two data reduction steps in Setup ICA when the number of subjects is greater than 10.
  • Two data reduction steps method is handled better in terms of memory usage for analyzing very large datasets.
  • Added C-MEX files for computing eigen values of a symmetric matrix using packed storage scheme. This approach is slower but uses less memory usage.
  • Added spatial-temporal back reconstruction approach.
