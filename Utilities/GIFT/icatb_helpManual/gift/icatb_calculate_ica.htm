ICA

The ICA algorithm chosen is an user-specified parameter (See Setup ICA). Depending upon the algorithm chosen, ICA options window pops up showing the options available to that algorithm. For explanation of some of the algorithms implemented in GIFT and how ICA options effect the ICA process see section 3.2 in the manual.

The reduced data-set from the data reduction step is used and ICA is applied to reveal independent components. The components (see Figure 1) represent group components.
