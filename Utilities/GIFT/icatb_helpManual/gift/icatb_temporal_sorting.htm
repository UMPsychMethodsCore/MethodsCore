Temporal Sorting

Temporal sorting is a way to compare the model's time course with the ICA time course. GIFT provides three ways of entering the regressors. The method of entering regressors is determined by the global variable METHOD_ENTERING_REGRESSORS in "icatb_defaults.m" file. The values for that variable are as follows:

  • 'AUTOMATIC' - Regressors are pulled from 'SPM.mat' file. Session related regressors will be used for 'Different regressors for sessions' or 'Different regressors for subjects and sessions'.
  • 'BATCH' - Regressors can be entered through a text file by specifying the path in TXTFILE_REGRESSORS variable. This way of entering the regressors is useful for large data-sets and each subject having a separate design matrix.
  • 'GUI' - GUI will open to select the regressors.

Select 'Temporal' option in Sorting GUI and use SPM design matrix for the model time course. Figure 2 shows the sorted components using Multiple Linear Regression sorting criteria. Here right and left visual are selected as the model time courses (Figure 1) for the visuomotor task (See Experimental Paradigms ). The time course window can be viewed in a separate figure (Expanded view of time course) if clicked on the time course plot. Expanded view of time course window has features like event average, splitting time courses, adjusting ICA time course and Fast Fourier Transform.
