Write Talairach Tables

We use Talairach Daemon software for generating talairach tables. When you select "Write Talairach Table" option in Utilities drop down box, a figure window will open to select the image. We apply a threshold from variable TALAIRACH_THRESHOLD in 'icatb_defaults.m' and write talairach coordinates for positive and negative regions in separate excel files.

Note: If you are using the mean or individual subject component images from the GIFT directly, the default units of the components will be in percent signal change unless you have selected a different scaling method such as Z-scores.
