Running GIFT

 
GIFT consists of mainly analysis functions and visualization methods. First analysis functions are explained followed by the visualization methods.

Analysis Functions:

Analysis functions are divided into three categories like Setup ICA Analysis, Run Analysis and Analysis Info.

Setup ICA Analysis:

  • First click on the button Setup ICA Analysis and set up parameters for group ICA analysis.

  • Once the parameters are set up, all the information is stored in a parameter file. The parameter file will be shown in the MATLAB command window.

Run Analysis:

  • Run analysis button is used for performing the group ICA analysis.

  • When the analysis is done all the information is stored in the same parameter file.

Analysis Info:

  • Analysis Info is used for displaying the analysis information. It uses the parameter file after the run analysis step is completed.

  • You can see the information about the parameters, data reduction and output files.

Visualization Methods:

  • Visualization methods consists of novel component visualization options  like Component Explorer, Orthogonal Viewer and Composite Viewer.

  • GIFT provides the user an interactive way to visualize components in the form of Display GUI.

  • Display GUI loads the parameter file after the run analysis step is completed. Sorting option is provided in component explorer of Display GUI to determine the task related components.

Note: In all the main figure windows GIFT-Help menu is plotted. When you click GIFT-Help menu corresponding HTML page is opened in default browser.
