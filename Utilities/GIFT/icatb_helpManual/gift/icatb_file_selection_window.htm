File Selection Window

Interactive figure window can be used to select a directory, file or files for performing the group ICA. The steps involved are explained below where the button names are shown in italic for your convenience.

Figure 1: File selection window

Directory text box
Shows the current directory or you can edit the text box. When enter key is pressed the sub-folders are displayed on the left list box and also the files or folders are displayed on the right list box.
Directory History
Shows the history of the directories recently visited in the MATLAB along with the path for the GIFT. Option is provided in 'icatb_defaults.m' file to store the directories where data is located. If you wish not to store the information leave the default of the variable STORE_DIRECTORY_INFORMATION as 'No'. When the variable is changed to 'Yes' the directories entered in the curly brackets are stored in the figure window.
Back
Returns to the previous folder in the same drive.
Filter
Resets the filter text box to *.
Filter text box
Filters files based on the text entered. If you want to list files with different patterns, separate them with semi-colon as delimiter.
File numbers text box
Text box used to enter the file numbers for a 4D Nifti file.
Drives
Shows the drives available for the Windows operating system. For other operating systems the root directory is displayed.
Sub-folders list box
Sub-folders are shown in the left list box.
Files or sub-folders list box
Files in the current directory are shown in the right list box with the number of files listed in square brackets like [1:20] where 20 refers to the number of files for that prefix. In case of directory selection, sub-folders in the current directory will be displayed.
Ed
Button used to edit the files
Reset
Refreshes the drives in the Windows operating system and deselects all the files previously selected.
Expand
By default files are shown in prefix mode. Expand button is used to list files with their original names.
Select-All
Selects all the files in the current working directory depending on the filter specified.
OK
Once the entries on the right list box are selected, press OK to select the entries.

Note: When a directory or a ICA parameter file is to be selected, the right list box mode is set to inactive after the first selection. In order to select a different file or directory Reset button should be used.
