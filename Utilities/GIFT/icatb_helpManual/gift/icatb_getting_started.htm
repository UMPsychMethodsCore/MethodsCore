Installing GIFT

Before installing GIFT, download 'example subjects.zip' file and unzip into an appropriate directory. Included in this file are three subject's preprocessed fMRI data from a visuomotor task (See Appendix A for a complete description of the task). Whole brain and single slice data (for rapid testing) are provided.

Download 'GroupICATv2.0d.zip' file and unzip into an appropriate directory. Add icatb directory and its sub-folders to MATLAB search path. You can also add path when you run 'groupica.m' or 'gift.m' file and this will automatically add folders and sub-folders at the bottom of the MATLAB path. You can specify your own paths by creating a 'gift_startup.m' file. You can open GIFT toolbox (Figure 2) in two ways when you click on fMRI button (Figure 1) or enter groupica 'fmri'.
