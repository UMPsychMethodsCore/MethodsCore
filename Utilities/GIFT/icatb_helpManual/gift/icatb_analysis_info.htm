Analysis Info

  • Analysis Info contains the information about the parameters, data reduction and the output files.
  • Once the analysis is done, click on the Analysis Info button in the GIFT main window and select the parameter file that you want to look at. Figure below shows the analysis information.

Figure 1: Analysis Info

The number of scans shown in the figure is the number of files in the first data-set. After viewing the analysis information, click on Display GUI button to visualize the results.
