Spatial Sorting

Spatial sorting classifies the components by comparing the component's image with the template. Select a spatial template after selecting 'Spatial' option under 'Select Sorting Type' in figure sorting GUI.

Figure below shows the sorted components of subject 1 session 1 using Multiple Linear Regression sorting criteria. Here 'RightTemplate.nii' and 'LeftTemplate.nii' are selected as the templates for the visuomotor task (See Experimental Paradigms).

Figure 1: Sorted components using spatial sorting

Note: Spatial template must be in Analyze or Nifti format. Sample templates are provided in the folder icatb_templates after you download the gift zip file.
