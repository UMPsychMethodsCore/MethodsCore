Adjust ICA Time course

ICA time course can be adjusted by removing the nuisance parameters and model parameters other than the selected model time course. The selected reference function to adjust ICA time course is Sn(1) right*bf(1). The ICA time course shown in the expanded view is subtracted with the line fit of model with the data. The model contains the nuisance parameters and other than the selected reference function (Sn(1) left*bf(1)). After the time course is adjusted the new plot is shown in the same figure (Figure 1) and plot button should be used to return to the original time course plot.
