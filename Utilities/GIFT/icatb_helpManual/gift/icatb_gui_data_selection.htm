Data Selection Window

The explanation for each control in figure 1 is given below:

Data-sets List box
The data-sets are shown in a list box on the left hand side. When clicked on an entry, a figure window will open to select the data.
View
The files for the selected data set can be viewed by clicking on the View button.
Change
The data for that data-set can be changed by clicking on the Change button.
Help
Displays a dialog box explaining the functions of each button.
OK
After selecting all the files, Click on Ok button to proceed to the next process.
Text Box
Text box at the bottom shows the number of files selected for that entry.
